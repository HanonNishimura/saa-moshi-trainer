# SAA 模試トレーナー（PWA）

AWS 認定ソリューションアーキテクト アソシエイト（SAA-C03）の模擬試験を、スマホ／PCのブラウザで学習できる **PWA**（オフライン対応・インストール可能）です。

> **プライバシー**: 問題データはアプリに同梱していません。利用者が CSV を**取り込み**、その端末内（localStorage）にのみ保存されます。問題文がネットに送信・公開されることはありません。このリポジトリにもアプリ本体（HTML/CSS/JS）だけが含まれ、問題データは含まれません。

## 主な機能

- 📝 **試験モード** — 模試選択／章（ジャンル）絞り込み／正答率の低い問題に絞る／★ブックマークのみ／問題数・出題順を自由に組み合わせ。タイマー・採点・記録。
- 🔀 選択肢シャッフル（位置で覚えるのを防止）
- 📖 **解説モード／学習モード** — 1問ずつ正解＋解説。**Gemini解説**があれば主表示、元の解説は「詳しい解説」として展開。前回の自分の回答を色表示、キーワード/番号検索、絞り込み。
- 📊 **弱点分析** — ジャンル別正答率・正答率の推移・よく間違える問題。AI に渡せる学習データを書き出し。
- 💾 中断した試験の再開、データのバックアップ/復元、文字サイズ変更
- 📱 スワイプ操作（スマホ）／キーボード操作（PC）、PWA更新通知

## 取り込めるファイル

| 種類 | 例 | 必須 |
|---|---|---|
| 問題集 CSV | `模試②_問題集.csv` | ✅ |
| 章別チェックリスト CSV | `模試②_章別チェックリスト.csv` | 任意（ジャンル分類を精密化） |
| Gemini解説 md | `模試②_Gemini解説.md` | 任意（解説を強化） |

同じ模試のファイルをまとめて取り込むと自動で1デッキに統合されます。

## 使い方（ホスティング）

PWA としてインストールするには https または localhost が必要です。

- **GitHub Pages**: リポジトリの Settings → Pages → Branch を `main` / `root` に設定 → 発行された URL をスマホで開き「ホーム画面に追加」。
- **ローカル確認(PC)**: `起動(PC確認用).bat` をダブルクリック（要 Python）→ `http://localhost:8000`。

詳しい操作は `使い方.txt` を参照してください。
